## Data Structures in JS using POI and Jest (Depth First Search Example)

  Install [POI](https://poi.js.org/):

```bash
$ npm i -g poi
```

  Install Jest:

```bash
$ npm i -D jest
```

  Run App and open it in a Web browser:

```bash
$ poi --serve
```

  Test Depth First Search:

```bash
$ npm test
```
